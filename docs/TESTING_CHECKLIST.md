# 테스트 체크리스트

이 문서는 배포 전 전체 사용자 플로우를 점검하기 위한 체크리스트입니다.

## 🔐 인증 플로우

### 회원가입
- [ ] 이메일로 회원가입 가능
- [ ] 소셜 로그인 (Google 등) 작동
- [ ] 회원가입 후 자동 로그인
- [ ] 회원가입 후 홈페이지로 리다이렉트

### 로그인
- [ ] 이메일/비밀번호 로그인 작동
- [ ] 소셜 로그인 작동
- [ ] 로그인 후 이전 페이지로 리다이렉트
- [ ] 로그인 상태 유지 (새로고침 후에도)

### 로그아웃
- [ ] 로그아웃 버튼 작동
- [ ] 로그아웃 후 홈페이지로 이동

## 🛍️ 쇼핑 플로우

### 상품 탐색
- [ ] 홈페이지 정상 로드
- [ ] 상품 목록 페이지 정상 표시
- [ ] 카테고리 필터 작동
- [ ] 정렬 기능 작동 (최신순, 가격순 등)
- [ ] 페이지네이션 작동
- [ ] 검색 기능 작동 (있는 경우)

### 상품 상세
- [ ] 상품 상세 페이지 정상 표시
- [ ] 상품 정보 (이름, 가격, 설명, 재고) 정확히 표시
- [ ] 장바구니 담기 버튼 작동
- [ ] 재고 부족 시 품절 표시

### 장바구니
- [ ] 장바구니 아이콘에 아이템 개수 표시
- [ ] 장바구니 페이지 정상 표시
- [ ] 수량 변경 작동
- [ ] 아이템 삭제 작동
- [ ] 총 금액 정확히 계산

## 💳 주문 & 결제 플로우

### 주문 생성
- [ ] 배송지 정보 입력 폼 작동
- [ ] 폼 유효성 검사 작동
- [ ] 주문 생성 성공
- [ ] 주문 생성 후 결제 페이지로 이동

### 결제
- [ ] 결제 위젯 정상 로드
- [ ] 결제 수단 선택 가능
- [ ] 결제 요청 성공
- [ ] 결제 성공 후 주문 상태 업데이트
- [ ] 결제 실패 시 실패 페이지 표시

### 주문 완료
- [ ] 결제 성공 후 주문 상세 페이지 표시
- [ ] 주문 정보 정확히 표시
- [ ] 배송지 정보 정확히 표시
- [ ] 주문 상품 목록 정확히 표시

## 👤 마이페이지

### 주문 내역
- [ ] 마이페이지 접근 가능
- [ ] 주문 내역 목록 정상 표시
- [ ] 주문 상태별 배지 색상 구분
- [ ] 주문 상세 페이지 이동 작동

### 주문 상세
- [ ] 주문 상세 정보 정확히 표시
- [ ] 주문 상품 목록 정확히 표시
- [ ] 배송지 정보 정확히 표시

## 🐛 에러 처리

### 네트워크 에러
- [ ] 네트워크 오류 시 적절한 에러 메시지 표시
- [ ] 재시도 기능 제공

### 인증 에러
- [ ] 인증되지 않은 사용자 접근 시 로그인 페이지로 리다이렉트
- [ ] 세션 만료 시 적절한 처리

### 데이터 에러
- [ ] 존재하지 않는 상품 접근 시 404 페이지 표시
- [ ] 존재하지 않는 주문 접근 시 404 페이지 표시
- [ ] 빈 상태 (장바구니, 주문 내역) 적절히 처리

## 📱 반응형 디자인

### 모바일 (375px)
- [ ] 레이아웃 정상 표시
- [ ] 네비게이션 메뉴 작동
- [ ] 폼 입력 정상 작동
- [ ] 버튼 클릭 영역 충분

### 태블릿 (768px)
- [ ] 레이아웃 정상 표시
- [ ] 그리드 레이아웃 적절히 조정

### 데스크톱 (1024px+)
- [ ] 레이아웃 정상 표시
- [ ] 최대 너비 제한 적용

## ⚡ 성능

### 로딩 속도
- [ ] 초기 로드 시간 < 3초
- [ ] 페이지 전환 부드러움
- [ ] 이미지 로딩 최적화

### 사용자 경험
- [ ] 로딩 스켈레톤 표시
- [ ] 버튼 클릭 시 피드백 제공
- [ ] 폼 제출 시 로딩 상태 표시

## 🔒 보안

### 인증
- [ ] 인증되지 않은 사용자는 보호된 페이지 접근 불가
- [ ] 사용자별 데이터 격리 (자신의 주문만 조회)

### 데이터 검증
- [ ] 서버 사이드 데이터 검증
- [ ] 클라이언트 사이드 입력 검증
- [ ] SQL 인젝션 방지 (Supabase 사용)

## 📝 체크리스트 사용 방법

1. 각 항목을 실제로 테스트하며 체크
2. 실패한 항목은 버그로 기록
3. 버그 수정 후 다시 테스트
4. 모든 항목이 체크되면 배포 준비 완료
